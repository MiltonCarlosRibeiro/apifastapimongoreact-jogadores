🛠️ Guia de Reinstalação Pós-Compactação
Sempre que você descompactar o projeto em uma pasta nova ou outro PC, siga este ritual:

1. No Backend (O Cérebro)
As bibliotecas Python não viajam no ZIP. Você precisa recriar o ambiente isolado:

Criar Venv: python3 -m venv .venv

Instalar Motores: source .venv/bin/activate && pip install fastapi uvicorn motor pymongo

2. No Banco (A Memória)
O MongoDB precisa de um lugar físico para respirar:

Diretório: Garanta que a pasta ./data existe na raiz ou use o serviço do sistema.

Comando: sudo systemctl start mongod (e sempre limpe o .lock se ele não subir).

3. No Frontend (O Rosto)
O React é pesado e precisa baixar as peças novamente:

Instalar Tudo: npm install (dentro da pasta frontend).

Start: npm start (para rodar na porta 3000).

💡 Lição de Ouro do Engenheiro
"O código viaja no ZIP, mas as dependências nascem no destino."
